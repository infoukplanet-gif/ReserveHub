# デザイン仕様書

## デザインコンセプト

**「シンプルで迷わない、でもプロっぽい」**

ターゲット: ITに強くない中小事業者 → 迷わせない、説明なしで使える
差別化: 料金設定の柔軟性 → 複雑な設定を「簡単そうに見せる」のが勝負

## アニメーション・インタラクション

```
ページ遷移:   なし（高速遷移を優先）
ホバー:       shadow-sm → shadow-md（150ms ease）
モーダル:     fade + slide up（200ms ease-out）
Sheet:        slide right（200ms ease-out）
Toast:        slide in from top right（300ms）
ステッパー:    数字のscale（200ms）
料金更新:     数字のfade（150ms）← 料金がリアルタイムで変わるのを視覚的に
ローディング:  Skeleton（shadcn/ui デフォルト）
```

## テンプレートシステム（事業者HP）

### 初期テンプレート（5種類）

| テンプレート | テイスト | 主な対象 |
|---|---|---|
| Simple | ミニマル、余白多め、白基調 | 万能 |
| Modern | ダーク+アクセント、幾何学的 | ジム、ピラティス |
| Natural | アースカラー、丸み、手書き風 | サロン、整体 |
| Cool | モノトーン+ブルー、シャープ | クリニック、ビジネス |
| Warm | オレンジ〜ブラウン、温かみ | カフェ、飲食 |

### カスタマイズ可能項目
- メインカラー（カラーピッカー）
- サブカラー（カラーピッカー）
- ロゴ画像
- ヒーロー画像
