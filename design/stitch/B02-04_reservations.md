# B-2: 予約カレンダー / B-3: 予約リスト / B-4: 予約詳細

## B-2: 予約カレンダー

### 画面概要
スタッフ別の予約をカレンダーで可視化。事業者が最も頻繁に使う画面。

### レイアウト

#### ヘッダーバー
- 左: 「予約管理」H1
- 中央: ◀ 2026年4月5日(土) ▶（日付ナビ）
- 右: 表示切替ボタン [日][週][月][リスト]
- 右端: [+ 手動予約] ボタン（Primary）

#### スタッフフィルタ
- チップ形式: [全員] [山田花子] [佐藤健太] [鈴木美咲]

#### カレンダー本体（日表示）
```
           山田花子        佐藤健太        鈴木美咲
 09:00  │              │              │              │
 09:30  │              │              │              │
 10:00  │┌──────────┐│              │              │
 10:30  ││ 山田太郎   ││              │              │
 11:00  ││ 60分コース ││              │┌──────────┐│
 11:30  │└──────────┘│┌──────────┐││ 高橋遼    ││
 12:00  │              ││ 佐藤花子   ││ 45分コース ││
 12:30  │              ││ 90分コース ││            ││
 13:00  │┌──────────┐│└──────────┘│└──────────┘│
 13:30  ││ 田中一郎   ││              │              │
 14:00  ││ 60分コース ││              │              │
 14:30  │└──────────┘│              │              │
```

- 予約ブロック: Blue背景（確定）、Green背景（完了）、Red背景（キャンセル）
- ブロックをクリック → 予約詳細シート（B-4）が右から開く
- 空き枠をクリック → 手動予約フォーム

#### 週表示
- 横軸: 月〜日
- 縦軸: 時間
- 各セルに予約件数 or 小さなブロック

#### 月表示
- カレンダーの各日に予約件数のバッジ表示

## B-3: 予約リスト

### レイアウト

#### フィルタバー
- ステータス: [すべて] [確定] [完了] [キャンセル] [ノーショー]
- スタッフ: ドロップダウン
- 期間: 日付レンジピッカー
- 検索: 顧客名で検索

#### テーブル
```
│ 日時           │ 顧客名    │ メニュー      │ スタッフ  │ 金額     │ ステータス │
│ 4/5 10:00     │ 山田太郎  │ 60分コース    │ 山田花子  │ ¥12,000 │ [確定]    │
│ 4/5 11:30     │ 佐藤花子  │ 90分コース    │ 佐藤健太  │ ¥11,000 │ [確定]    │
│ 4/4 14:00     │ 田中一郎  │ 60分コース    │ 鈴木美咲  │ ¥8,000  │ [完了]    │
│ 4/3 16:00     │ 鈴木美咲  │ フェイシャル45分│ 山田花子  │ ¥6,000  │ [完了]    │
│ 4/2 10:00     │ 高橋遼   │ 60分コース    │ 佐藤健太  │ ¥8,000  │ [キャンセル]│
```

- 行クリック → 予約詳細シート
- ページネーション: 50件/ページ
