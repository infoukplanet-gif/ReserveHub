# A-2: メニュー・料金一覧

## 画面概要
事業者の全メニューをカテゴリ別に表示。料金の柔軟性を「わかりやすく」見せるのが重要。

## レイアウト

### ヘッダー（A-1共通）

### ページ見出し
- 「メニュー・料金」H1

### カテゴリタブ
- 横並びタブ: 「すべて」「ボディケア」「フェイシャル」「オプション」
- モバイル: 横スクロール

### メニューカード（カテゴリごとに）
各メニューはカード形式:
```
┌────────────────────────────────┐
│ [メニュー画像]                    │
│ ボディケア60分コース               │ ← メニュー名
│ 肩こり・腰痛にお悩みの方に         │ ← 説明
│ 所要時間: 60分                    │
│                                  │
│ ¥8,000〜¥10,000                 │ ← 料金範囲
│ 平日 ¥8,000 / 土日祝 ¥10,000    │ ← 料金詳細（小さめ）
│                                  │
│ オプション:                       │
│  ・ヘッドスパ追加 +¥1,500 (+15分) │
│  ・アロマ変更 +¥500              │
│                                  │
│ [このメニューで予約する]           │ ← Primaryボタン
└────────────────────────────────┘
```

### 回数券セクション
- 見出し: 「お得な回数券」
- カード形式:
```
┌──────────────────┐
│ 🎫 10回券         │
│ 対象: 60分コース   │
│ ¥70,000（1回¥7,000）│
│ 有効期限: 6ヶ月    │
│ [詳しく見る]       │
└──────────────────┘
```

### 料金に関する注意書き
- 「※料金は曜日・時間帯により異なります。詳細は予約画面でご確認ください。」
- 「※指名料が別途かかる場合があります。」

## サンプルメニュー
1. ボディケア60分 ¥8,000〜
2. ボディケア90分 ¥11,000〜
3. フェイシャル45分 ¥6,000〜
4. プレミアム全身120分 ¥15,000〜
