# B-5: メニュー管理 / B-6: メニュー編集

## B-5: メニュー管理（一覧）

### ヘッダーバー
- 左: 「メニュー管理」H1
- 右: [+ メニューを追加] ボタン（Primary）

### カテゴリタブ
- [すべて] [ボディケア] [フェイシャル] [セットメニュー]
- 右端: [カテゴリ管理] テキストリンク

### メニューカード（リスト形式）
```
┌──────────────────────────────────────┐
│ [画像] ボディケア60分コース             │
│        60分 + バッファ15分              │
│        ¥8,000〜¥10,000               │
│        オプション: 3個                  │
│        対応スタッフ: 3名                │
│        ステータス: [公開中]              │
│                         [編集] [⋮]    │ ← ⋮で複製・非公開・削除
├──────────────────────────────────────┤
│ [画像] ボディケア90分コース             │
│        ...                            │
└──────────────────────────────────────┘
```

- ドラッグ&ドロップで並び替え可能（☰ハンドル）

## B-6: メニュー編集（★最重要管理画面）

### 画面概要
料金ルール設定が差別化の核心。複雑な設定を「簡単そう」に見せるUIが勝負。

### レイアウト

#### パンくず
← メニュー管理 / ボディケア60分コース

#### 基本情報セクション
```
メニュー名 *
[ボディケア60分コース                    ]

カテゴリ *
[ボディケア                          ▼]

説明
[肩こり・腰痛にお悩みの方におすすめ...    ] ← Textarea

メニュー画像
[画像をアップロード] または ドラッグ&ドロップ

所要時間 *        バッファ時間
[60] 分           [15] 分
                  ※前後の準備時間
```
