# C-1: ログイン / C-2: 新規登録 / C-3: パスワードリセット

## C-1: ログイン

### レイアウト（中央配置、白カード）
```
┌─────────────────────────────┐
│                               │
│      [ReserveHub ロゴ]        │
│                               │
│    事業者アカウントにログイン   │ ← H2
│                               │
│    [G Googleでログイン]       │ ← Outlineボタン、フル幅
│                               │
│    ────── または ──────       │ ← Divider with text
│                               │
│    メールアドレス              │
│    [                      ]   │
│                               │
│    パスワード                  │
│    [                      ]   │
│    [パスワードを忘れた方はこちら]│ ← テキストリンク（右寄せ）
│                               │
│    [ログイン]                  │ ← Primaryボタン、フル幅
│                               │
│    アカウントをお持ちでない方   │
│    [新規登録はこちら]          │ ← テキストリンク
│                               │
└─────────────────────────────┘
```

### 背景
- フラット背景: #F8FAFC
- 装飾なし、または極薄の幾何学パターン（opacity 0.03のドット or グリッド）

## C-3: パスワードリセット

```
┌─────────────────────────────┐
│                               │
│      [ReserveHub ロゴ]        │
│                               │
│    パスワードをリセット        │ ← H2
│    登録メールアドレスに        │
│    リセットリンクを送信します   │ ← Small、Muted
│                               │
│    メールアドレス              │
│    [                      ]   │
│                               │
│    [リセットリンクを送信]       │ ← Primaryボタン、フル幅
│                               │
│    [← ログインに戻る]          │
│                               │
└─────────────────────────────┘
```

### 送信後の状態
```
┌─────────────────────────────┐
│                               │
│      ✅ 送信しました           │
│                               │
│    入力されたメールアドレスに   │
│    リセットリンクを送信しました。│
│    メールをご確認ください。     │
│                               │
│    [ログインに戻る]            │
│                               │
└─────────────────────────────┘
```
