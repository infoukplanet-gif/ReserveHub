# B-1: ダッシュボード

## 画面概要
事業者が毎日最初に見る画面。本日の予約状況と売上をひと目で把握。

### メインコンテンツ

#### 統計カード（4列、モバイルは2×2グリッド）
```

│ 本日   │ │ 今月   │ │ 新規   │ │リピート │
│ の予約  │ │ 売上   │ │ 顧客   │ │ 率     │

```
