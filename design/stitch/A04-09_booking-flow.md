# A-4〜A-9: 予約フロー（6ステップ）

## 画面概要
予約の全フロー。ステッパーで進捗を表示。料金がリアルタイムで更新されるのが差別化ポイント。
全6画面を連続したフローとして生成。

## 共通レイアウト

### ヘッダー
- 左: ← 戻るボタン + 「予約」
- 右: 店舗名

### ステッパー（画面上部）
- 6ステップの水平ステッパー
- ①メニュー → ②オプション → ③スタッフ → ④日時 → ⑤情報 → ⑥確認
- 完了ステップ: Blue丸 + チェック
- 現在ステップ: Blue丸 + 番号（大きめ）
- 未到達: Gray丸 + 番号
- ステップ間は線で接続

### 料金サイドバー（Desktop: 右固定 / Mobile: 下部固定バー）
```
┌─ 予約内容 ────────────┐
│ 60分ボディケアコース    │
│                ¥8,000  │
│ ヘッドスパ追加  +¥1,500 │
│ 指名(山田花子)  +¥500  │
│─────────────────────│
│ 合計         ¥10,000  │ ← 大きく太字
│                        │
│ [次へ進む]              │ ← Primaryボタン
└────────────────────┘

モバイル下部固定バー:
┌──────────────────────────┐
│ 合計 ¥10,000    [次へ進む] │
└──────────────────────────┘
```

## A-7: Step4 日時選択（★最重要画面）

### コンテンツ
- 見出し: 「日時を選ぶ」

### カレンダー
- 月表示のカレンダー（◀ 2026年4月 ▶）
- 過去日: Gray、選択不可
- 本日: 太字下線
- 休業日: ×マーク、Gray
- 選択日: Blue背景、白文字
- 土曜: 青文字、日祝: 赤文字

### 空き時間枠リスト（カレンダーの下）
- 選択した日付の見出し: 「4月5日(土)の空き時間」
- 時間枠はボタン形式:

```
┌────────────────────────────┐
│  10:00    ¥10,000    [選択] │ ← 休日料金が反映されている
├────────────────────────────┤
│  11:30    ¥10,000    [選択] │
├────────────────────────────┤
│  13:00    ¥10,000    [選択] │
├────────────────────────────┤
│  14:30    ¥10,000    [選択] │
├────────────────────────────┤
│  16:00    ¥10,000    [選択] │
└────────────────────────────┘
```

- 選択済み時間枠: Blue背景、白文字
- ★重要: 「平日料金」「休日料金」等のラベルを時間枠の横に小さく表示（料金が変わる理由を顧客に明示）
